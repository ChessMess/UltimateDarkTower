Depend on `ultimatedarktowerdata` directly for `GLYPHS`, `TOWER_LIGHT_SEQUENCES`, and
`TOWER_AUDIO_LIBRARY` — these moved out of `ultimatedarktower` in its v6.0.0 and are no longer
re-exported from the driver. No API change on this package's side; the same names are still
available from the same places (`TowerStateReadout`, the audio modules, the sequence modules).

In `TOWER_AUDIO_LIBRARY`, the autogenerated `audioLibrary.ts` sample map picks up two key renames
from core: `IsatheHollow(Spawn)` → `IsatheExile(Spawn)` and `Strigas` → `Striga` (spelling fixes,
no firmware change — see `ultimatedarktower`'s changelog).
