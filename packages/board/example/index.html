Click a token in either view to select it. Use the Palette to add one (Add → pick a space → Confirm) and the Inspector to edit the selection; drag panels by their title bar. Each view's N/E/S/W bar zooms to a kingdom (All restores the whole board). On the 2D map: scroll to zoom, drag to spin the board (flip to Pan with the Spin/Pan toggle), double-click to reset. The right column has the live board JSON — edit it and hit Apply, or Copy it for your own app.
