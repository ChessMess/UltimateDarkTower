Ultimate Dark Tower Board

Live demo — the 3D board (a Display ScenePlugin) + the 2D overhead map + the text readout + the dockable editing UI (palette / inspector / summary), all sharing one board state, selection, and focus. Click a token in either view to select it; use the palette to add one (click a space on the board, then Confirm); drag the panels to reposition them; N/E/S/W moves the camera and narrows the map/readout.
