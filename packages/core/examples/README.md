# Examples

The library's browser demos — **Controller** and **Game** — have been promoted to
standalone Vite apps in the monorepo's `apps/` directory. This folder now holds
only the Node CLI example.

| Example                                              | Runtime                 | Demonstrates                                                 |
| ---------------------------------------------------- | ----------------------- | ------------------------------------------------------------ |
| **[Controller](../../../apps/controller/README.md)** | Browser (Web Bluetooth) | Full command surface, BLE diagnostics tab, 3D tower emulator |
| **[Game](../../../apps/game/README.md)**             | Browser (Web Bluetooth) | A complete playable game on top of the library               |
| **[Node CLI](node/README.md)**                       | Node.js                 | Minimal command-line driver for the Node adapter             |

For a feature-by-feature breakdown of each example and how to run them, see
[docs/EXAMPLES.md](../docs/EXAMPLES.md).
