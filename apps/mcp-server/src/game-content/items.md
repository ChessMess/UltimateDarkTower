# Return to Dark Tower — Items

## Overview

There are four types of items: **Potions**, **Gear**, **Treasures**, and **Quest Items**. You gain items by Reinforcing, defeating foes, exploring dungeons, and completing quests. You can only use your items on your own turn.

---

## Item Types

### Potions

- **Capacity:** Unlimited
- **How to gain:** Draw the top card of the potion deck (Reinforce at a citadel, haggle die result, quest rewards, etc.)
- **When spent or lost:** Return to the bottom of the potion deck.
- If no potions remain in the deck, you can take a potion of your choice from another hero (competitive mode).

### Gear

- **Capacity:** Up to one of each type (6 types available)
- **How to gain:** Choose 1 card from the gear stacks (Reinforce at a bazaar, haggle die result, etc.)
- **When spent or lost:** Return to the matching gear stack.
- If you gain a duplicate gear, you must lose one.
- Gear often provides **automatic Advantages** (black background).

### Treasures

- **Capacity:** Up to four
- **How to gain:** Choose 1 of the face-up treasures from the market (Reinforce at bazaar enhanced effect, quest rewards, etc.)
- **When spent or lost:** Return to the bottom of the treasure deck.
- When a treasure leaves the market, replace it with the top card of the treasure deck.
- If you gain a 5th treasure, you must lose one.

### Quest Items

- **Capacity:** Unlimited
- **How to gain:** Find and take from the quest item deck in the box (quest rewards, dungeon completion, etc.)
- **When spent or lost:** Return to the game box.

---

## Item Capacity Summary

| Item Type  | Carry Limit       |
| ---------- | ----------------- |
| Potion     | Unlimited         |
| Gear       | Up to one of each |
| Treasure   | Up to four        |
| Quest Item | Unlimited         |

---

## Spending vs. Losing Items

- **Spending** is optional and can be done at any point during your turn. When you spend an item for a different effect (e.g., activating a virtue), you do **not** also get the item's normal effect.
- **Losing** is mandatory. If you cannot resolve the loss fully, lose as much as you can and gain a single corruption.

---

## When Spent or Lost — Return To:

| Item Type  | Returns To                  |
| ---------- | --------------------------- |
| Warriors   | The general supply          |
| Spirit     | The general supply          |
| Potion     | Bottom of the potion deck   |
| Gear       | The matching gear stack     |
| Treasure   | Bottom of the treasure deck |
| Quest Item | The game box                |

---

## Gaining Items During Battle/Dungeon

- If you gain an item in the middle of a battle or dungeon, you gain any **automatic Advantages** it gives and can use them immediately.
- If you lose an item that gave you an automatic Advantage, you **don't** lose that Advantage for the current action.
