# Return to Dark Tower — Buildings

## Overview

There are four types of buildings in each kingdom: **Citadel**, **Sanctuary**, **Village**, and **Bazaar**. Each kingdom has one of each type (16 buildings total).

## Reinforce Action

While on a space with a building, you may take the **Reinforce action** (once per turn). Each building offers a **free effect** and an **enhanced effect** that costs spirit. You choose one.

Before Reinforcing, you may optionally roll the **haggle die** (see below).

---

## Building Types

### Citadel

| Effect       | Description                                                                             |
| ------------ | --------------------------------------------------------------------------------------- |
| **Free**     | Gain 1 potion. Draw the top card of the potion deck.                                    |
| **Enhanced** | Spend spirit to gain one virtue. Choose 1 of your inactive virtues and flip it face up. |

### Sanctuary

| Effect       | Description                                                                                        |
| ------------ | -------------------------------------------------------------------------------------------------- |
| **Free**     | Gain 1 spirit. Take from the supply.                                                               |
| **Enhanced** | Spend spirit to remove **all** your corruptions. Return them to the bottom of the corruption deck. |

### Village

| Effect       | Description                                                     |
| ------------ | --------------------------------------------------------------- |
| **Free**     | Gain 2 warriors. Take from the supply.                          |
| **Enhanced** | Spend spirit to gain additional warriors. Take from the supply. |

### Bazaar

| Effect       | Description                                                              |
| ------------ | ------------------------------------------------------------------------ |
| **Free**     | Gain 1 gear. Choose 1 card from the gear stacks.                         |
| **Enhanced** | Spend spirit to gain 1 treasure. Choose 1 card from the treasure market. |

---

## Building Destruction

- Each building can hold up to **3 skulls**.
- If a **4th skull** would be placed on a building, the building is **destroyed**.
- When destroyed: the building and its 3 skulls are **removed from the game**.
- The 4th skull returns to the supply.
- The hero whose home kingdom the building was in **gains a corruption**.
- If the destroyed building was in a **dormant kingdom**, no one gains a corruption.

---

## Haggle Die

Before Reinforcing, you may optionally roll the haggle die. Results:

| Result   | Effect                               |
| -------- | ------------------------------------ |
| Blank    | Reinforce normally                   |
| Warriors | After you Reinforce, gain 3 warriors |
| Potion   | After you Reinforce, gain 1 potion   |
| Gear     | After you Reinforce, gain 1 gear     |
| Cancel   | The Reinforce action is **canceled** |

---

## Skull Placement Rules

When skulls emerge from the Tower:

- Place them on buildings in the kingdom where they first emerged.
- Each player decides where to place skulls in their home kingdom.
- The player who dropped the skull decides for dormant kingdoms.
